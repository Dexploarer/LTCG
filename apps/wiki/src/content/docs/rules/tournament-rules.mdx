This document outlines the official rules for LTCG organized play events.

## General Rules

### Player Eligibility

To participate in official tournaments:

- Must have a valid LTCG account
- Must be at least 13 years old
- Must agree to Tournament Terms of Service
- Some events require minimum rank

### Account Requirements

| Requirement | Details |
|-------------|---------|
| Email Verified | Yes |
| Phone Verified | Regional events+ |
| Rank | Event-specific |
| Standing | No active bans |

## Deck Rules

### Standard Format

| Rule | Requirement |
|------|-------------|
| Deck Size | 45-60 cards |
| Copy Limit | 3 per card |
| Banned Cards | 0 allowed |
| Limited Cards | 1 copy max |
| Sideboard | 0-15 cards (if applicable) |

### Deck Submission

- Submit deck before tournament start
- Lock-in deadline is strictly enforced
- No changes after submission
- Deck codes verified automatically

### Deck List Verification

For high-stakes events:
- Decks may be manually verified
- Random deck checks during event
- Discrepancies result in game loss or disqualification

---

## Match Procedures

### Before the Match

1. Check opponent assignment
2. Verify deck is locked
3. Accept match within time limit
4. Connect to game server

### Starting the Game

- Higher seed chooses play/draw (elimination rounds)
- Random determination (Swiss rounds)
- 60 seconds to mulligan decision
- 5-card starting hand

### During the Game

- 90 seconds per turn (default)
- No external assistance
- No communication with spectators
- Screen capture may be required

### Time Limits

| Format | Per Turn | Per Match |
|--------|----------|-----------|
| Bo1 | 90 sec | 15 min |
| Bo3 | 90 sec | 45 min |
| Conquest | 90 sec | 50 min |

### End of Match

- Winner reports automatically
- Disputes escalate to judge
- Next match begins after 5-minute break

---

## Match Formats

### Best of 1 (Bo1)

- Single game decides match
- No sideboard
- Used in Swiss rounds

### Best of 3 (Bo3)

- First to win 2 games
- Sideboarding between games (if allowed)
- Used in elimination rounds

### Conquest

- Bring 3 different decks
- Must win with all 3 to win match
- Winning deck retired for the match

### Last Hero Standing

- Bring 3 different decks
- Keep winning deck, switch on loss
- Lose all 3 decks to lose match

---

## Sideboarding Rules

When sideboarding is allowed:

### Sideboard Requirements

- 0-15 cards in sideboard
- 45-60 cards in main deck
- Combined pool cannot exceed 75 cards

### Between Games

- 3 minutes to sideboard
- Any number of swaps allowed
- Deck must still be legal
- Sideboard changes are private

## Disconnections

### Player Disconnect

| Scenario | Resolution |
|----------|------------|
| Reconnect within 60 sec | Game continues |
| Reconnect within 2 min | Opponent choice: continue or restart |
| Fail to reconnect | Game loss |

### Server Issues

- Both players affected = match restart
- Tournament pause if widespread
- Judges determine fair resolution

### Disconnect Abuse

- 3+ disconnects in event = warning
- Intentional disconnect = game loss
- Pattern of abuse = disqualification

## Player Conduct

### Expected Behavior

- Treat opponents with respect
- Play in good faith
- Report issues promptly
- Follow judge rulings

### Prohibited Behavior

| Violation | Minimum Penalty |
|-----------|-----------------|
| Harassment | Disqualification |
| Cheating | Disqualification + ban |
| Account sharing | Disqualification |
| Match fixing | Permanent ban |
| Stalling | Warning → game loss |
| Unsportsmanlike conduct | Warning → DQ |

### Communication

- In-game chat is monitored
- Offensive language prohibited
- "GG" and basic courtesies encouraged
- Excessive messaging may be muted

---

## Cheating & Collusion

### Cheating Includes

- Using exploits or bugs intentionally
- Third-party assistance software
- Multiple accounts in same event
- Receiving outside help during match

### Collusion Includes

- Predetermined match outcomes
- Prize splitting agreements
- Intentional losing
- Win trading

### Reporting Cheating

If you suspect cheating:

1. Do not confront the player
2. Complete the match normally
3. Report through official channels
4. Provide evidence if available

---

## Judges & Appeals

### Judge Authority

Judges are the final authority on:
- Rule interpretations
- Penalty assignments
- Match results disputes
- Player conduct

### Appealing Decisions

1. Accept initial judge ruling
2. Request appeal to head judge
3. Head judge decision is final
4. Appeals must be immediate

### Contacting a Judge

- Use in-game judge call button
- Wait for judge response
- Do not continue match until resolved

## Prizes & Payouts

### Prize Distribution

- Prizes awarded based on final placement
- Must claim within 30 days
- Forfeited prizes redistributed

### Prize Eligibility

Cannot receive prizes if:
- Disqualified from event
- Discovered cheating (retroactive)
- Account in bad standing
