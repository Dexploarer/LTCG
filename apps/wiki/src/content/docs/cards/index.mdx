## Card Types

| Type | Description |
|------|-------------|
| Monster | Creatures that attack and defend |
| Spell | One-time effects |
| Trap | Set cards that trigger on conditions |
| Field | Continuous battlefield effects |
| Equipment | Attach to monsters for bonuses |
| Ritual | Special summon requirements |
| Fusion | Combine multiple monsters |

## Rarities

- **Common** - Basic cards, easy to obtain
- **Uncommon** - Slightly stronger effects
- **Rare** - Powerful archetype support
- **Epic** - High-impact effects
- **Legendary** - Iconic boss monsters
