Unranked mode offers a relaxed environment to play without the pressure of rank or rating.

## Overview

Unranked (Casual) mode is perfect for:
- Testing new decks
- Learning the game
- Playing with friends
- Completing quests without rank risk
- Having fun without stakes

---

## Matchmaking

### How It Works

Unranked uses a separate **Casual MMR** to match players:

- Hidden rating based on unranked win/loss
- Looser matching than ranked (faster queues)
- Attempts to match deck strength
- New players protected from veterans

### Queue Times

| Time of Day | Typical Queue |
|-------------|---------------|
| Peak hours | 5-15 seconds |
| Off-peak | 15-45 seconds |

---

## Casual vs Ranked

| Feature | Unranked | Ranked |
|---------|----------|--------|
| Rank changes | No | Yes |
| Win rewards | Yes | Yes |
| Quest progress | Yes | Yes |
| Seasonal rewards | No | Yes |
| Deck restrictions | Standard rules | Standard rules |
| Matchmaking | Looser | Stricter |
| Concede penalty | None | Star loss |

---

## Game Settings

### Match Rules

- Best of 1
- 90-second turn timer (same as ranked)
- No match timer (unlike ranked)
- Standard deck rules apply

### Deck Requirements

Same as ranked:
- Minimum 45 cards
- Maximum 3 copies of any card
- No banned cards

---

## Rewards

### XP and Gold

| Outcome | Gold | XP |
|---------|------|-----|
| Win | 50-80 | 100 |
| Loss | 10-20 | 30 |
| First Win Bonus | +100 | +50 |

### Quest Completion

All quests can be completed in unranked mode:
- "Win X games" - counts
- "Play X cards of type" - counts
- "Deal X damage" - counts

:::tip[Quest Farming]
Unranked is great for completing type-specific quests without risking rank!
:::

---

## Conceding

You can concede at any time in unranked:
- No rank penalty
- Opponent still gets full win rewards
- Game counts for quests (if requirements met)
- Counts toward daily play bonuses

### Concede Etiquette

- It's okay to concede unwinnable games
- Conceding for quests is normal
- Repeated instant-concedes may trigger cooldown

---

## Social Features

### Challenge Friends

In unranked, you can:
1. Challenge friends from your friends list
2. Create a private lobby with code
3. Spectate friends' unranked games

### Private Matches

Custom games between friends offer:
- No rewards (prevents boosting)
- Custom deck rules (optional)
- Best of 3 option
- Spectator mode

---

## Tips for Casual Play

1. **Test decks here first** - Before taking to ranked
2. **Learn new archetypes** - Low-stakes practice
3. **Complete tough quests** - Without rank anxiety
4. **Play meme decks** - Have fun with unusual builds
5. **Warm up** - Play a casual game before ranked session

---

## New Player Recommendation

If you're new to LTCG:

1. Complete the **Tutorial**
2. Play **Practice Mode** vs AI
3. Try **Unranked** to learn player strategies
4. Enter **Ranked** when comfortable

Unranked is always available as a low-pressure way to enjoy the game!
