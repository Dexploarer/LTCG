## Comparison Features

- **Side-by-side stats**: ATK, DEF, Level compared visually
- **Effect text**: Full effect descriptions for each card
- **Crafting cost**: See dust requirements at a glance
- **Rarity indicators**: Color-coded for quick reference

## Tips

- Compare cards within the same archetype to optimize your deck
- Use this to decide between similar cards for tech slots
- Check effects carefully - small differences can be significant
